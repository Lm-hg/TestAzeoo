# TestAzeoo - Flutter Skills Demonstration

This is a Flutter application created to demonstrate Flutter development skills. The project showcases various Flutter concepts and best practices.

## Features Demonstrated

- **StatefulWidget & State Management**: Interactive counter with increment, decrement, and reset functionality
- **Material Design 3**: Modern Material Design implementation with theming
- **Animated Widgets**: Smooth animations using AnimatedContainer
- **Responsive Layout**: Flexible UI that adapts to different screen sizes
- **Custom Styling & Theming**: Custom color schemes and typography
- **User Interactions**: Multiple interactive elements (buttons, icons, gestures)
- **Widget Composition**: Well-structured widget hierarchy

## Getting Started

### Prerequisites

- Flutter SDK (>=3.0.0)
- Dart SDK
- IDE (VS Code, Android Studio, or IntelliJ IDEA)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Lm-hg/TestAzeoo.git
cd TestAzeoo
```

2. Install dependencies:
```bash
flutter pub get
```

3. Run the application:
```bash
flutter run
```

### Running Tests

To run the widget tests:
```bash
flutter test
```

## Application Overview

The application features:

1. **Interactive Counter**: A counter that can be incremented, decremented, and reset
2. **Animated Shape**: An animated container that expands/collapses with smooth transitions
3. **Skills Display**: A card listing the Flutter skills demonstrated in the project
4. **Color Feedback**: The counter color changes based on the value (red when > 10)
5. **Multiple Action Buttons**: Different ways to interact with the counter

## Flutter Concepts Used

- StatefulWidget and StatelessWidget
- State management with setState()
- Material Design widgets (Scaffold, AppBar, Card, etc.)
- Layout widgets (Column, Row, Center, etc.)
- Animation with AnimatedContainer
- Theme management
- Icon usage and customization
- Responsive design patterns
- Widget testing with flutter_test
